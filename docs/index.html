プログラミングをやってみよう

Neumann Lab (ノイマンラボ) は静岡県三島市で活動している初心者・子供向けのプログラミング教室です。

  • まったくプログラミングをしたことがない人
  • 本を読んでプログラミングをしてみたけど挫折した人
  • プログラミング仲間がほしい人

誰でも参加できます❗ご相談ください。

大人も子供も同じ技術を学ぼう

みんなで同じ教材を使います
子供であっても大人と同じ技術を使ってプログラミングを学ぶべきです。
Pythonなどのプログラミング言語を学ぼう
Scratch のようなビジュアルプログラミングではなく Python などのテキストプログラミング言語を使います。
現役プログラマが教えます
実際に仕事でプログラミングを使っている講師が教えます。
